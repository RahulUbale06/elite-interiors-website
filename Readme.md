# Elite Interiors Website

A modern luxury interior designer website built using HTML, CSS, and JavaScript.

This project originally started as part of my frontend learning journey and basic web development practice during my diploma studies. Later, I redesigned and improved the UI/UX, layout structure, styling, and overall visual appearance to explore modern frontend development concepts and premium website aesthetics.

The project was also enhanced using AI-assisted frontend workflows to improve design quality, responsiveness ideas, and user interface structure.

- Modern luxury-style UI
